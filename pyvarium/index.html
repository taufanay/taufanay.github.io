PYVARIUM

Multipurpose Python Script Runner GUI
Home / Pyvarium

Pyvarium is a Python script runner GUI tool.
It designed to be easy and quickly load and execute scripts just like using command line.
It feature result that can be represented as textual or graphical using the PySide2 widgets to provide convenience and interactivity.
It was created as a testing and analitics tool in mind, but come up as a multipurpose tool.

This application is distributed in source code form.
To run it, you need at least Python3 and PySide2 environment already set up in your computer.

The documentations are included in the ./Collections/Welcome directory that can be opened inside this application, too!
I will explain the features and how to use it in my YouTube channel as this application progress.

Stay tune!
